# Legal & Compliance Specification

## Overview

"연정" 서비스의 법적 요구사항 및 규정 준수 스펙입니다.
한국 개인정보보호법(PIPA) 및 전자상거래법 기준으로 작성되었습니다.

## 2. 개인정보 수집 및 처리

### 2.1. 수집 항목

| 구분 | 항목 | 수집 목적 | 필수/선택 |
|------|------|----------|----------|
| **회원정보** | 이름, 이메일 | 계정 식별, 서비스 제공 | 필수 |
| | 프로필 이미지 | 프로필 표시 | 선택 |
| | OAuth ID | 소셜 로그인 연동 | 필수 |
| **청첩장** | 신랑/신부 이름 | 청첩장 생성 | 필수 |
| | 예식 일시/장소 | 청첩장 내용 | 필수 |
| | 연락처 | 연락하기 기능 | 선택 |
| | 계좌 정보 | 축의금 안내 | 선택 |
| | 혼주 정보 | 청첩장 내용 | 선택 |
| **RSVP** | 하객 이름 | 참석 확인 | 필수 |
| | 하객 연락처 | 감사 인사 발송 | 선택 |
| | 참석 여부/인원 | 참석 집계 | 필수 |
| **방명록** | 작성자명 | 메시지 표시 | 필수 |
| | 비밀번호 (해시) | 삭제 권한 | 필수 |

### 2.2. 보유 기간

| 데이터 | 보유 기간 | 근거 |
|--------|----------|------|
| 회원정보 | 탈퇴 시까지 | 서비스 제공 |
| 청첩장 데이터 | 예식일 + 1년 | 서비스 정책 |
| RSVP 데이터 | 예식일 + 1년 | 서비스 정책 |
| 방명록 | 예식일 + 1년 | 서비스 정책 |
| 결제 기록 | 5년 | 전자상거래법 |
| 접속 로그 | 3개월 | 통신비밀보호법 |

### 2.3. 파기 절차

```
1. 파기 사유 발생 (탈퇴, 보유기간 만료)
     ↓
2. 개인정보 보호책임자 승인
     ↓
3. 파기 방법 결정
   - DB: DELETE 쿼리 + 백업 파기
   - 파일: 영구 삭제
     ↓
4. 파기 완료 기록 (로그)
```

## 4. 동의 관리

### 4.1. 동의 수집 Flow

```
[회원가입]
     ↓
┌─────────────────────────────────────────────────────────────┐
│  연정 시작하기                                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [✓] (필수) 이용약관에 동의합니다                            │
│  [✓] (필수) 개인정보 수집 및 이용에 동의합니다                │
│  [ ] (선택) 마케팅 정보 수신에 동의합니다                     │
│                                                             │
│  ─────────────────────────────────────────────────────────  │
│                                                             │
│  [카카오로 시작하기]                                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2. 동의 이력 저장

```typescript
interface ConsentRecord {
  userId: string;
  type: 'terms' | 'privacy' | 'marketing';
  agreed: boolean;
  version: string;        // 약관 버전
  agreedAt: Date;
  ipAddress: string;
  userAgent: string;
}
```

### 4.3. 동의 철회

| 동의 유형 | 철회 방법 | 영향 |
|----------|----------|------|
| 이용약관 | 회원 탈퇴만 가능 | 서비스 이용 불가 |
| 개인정보 | 회원 탈퇴만 가능 | 서비스 이용 불가 |
| 마케팅 | 설정에서 언제든 | 마케팅 수신 중단 |

## 5. 정보주체 권리 보장

### 5.1. 권리 행사 방법

| 권리 | 방법 | 처리 기간 |
|------|------|----------|
| 열람 요청 | 마이페이지 또는 고객센터 | 즉시 또는 10일 이내 |
| 정정 요청 | 마이페이지 | 즉시 |
| 삭제 요청 | 회원 탈퇴 | 즉시 (백업 제외) |
| 처리정지 | 고객센터 | 10일 이내 |

### 5.2. 셀프서비스

```
마이페이지 > 개인정보 관리
├─ 내 정보 보기 (열람)
├─ 정보 수정 (정정)
├─ 마케팅 수신 설정 (동의 철회)
└─ 회원 탈퇴 (삭제)
```
