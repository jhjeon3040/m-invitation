# Product Strategy & Differentiation

## 1. Brand Identity

### 1.1. Brand Name & Meaning
- **Name**: 연정 (戀情, Yeonjeong)
- **Meaning**: 사랑하는 마음, 연인 사이의 따뜻한 감정
- **Tagline**: "사랑의 시작을 담다"

### 1.2. Brand Personality
| Attribute | Description |
|-----------|-------------|
| **Warm** | 따뜻하고 포근한 감성 |
| **Romantic** | 설레고 아름다운 순간 |
| **Elegant** | 절제된 고급스러움 |
| **Personal** | 커플만의 개성을 존중 |

### 1.3. Design Philosophy: "Dreamy Romantic Editorial"
세 가지 감성의 조화:
1. **Bright Romantic**: 밝고 따뜻한 파스텔 톤의 로맨틱 감성
2. **Simple Minimal**: 불필요한 요소를 덜어낸 절제된 아름다움
3. **Artistic Editorial**: 매거진/에디토리얼 스타일의 세련된 레이아웃

## 2. Market Analysis

### 2.1. Pain Points (문제점)
| Problem | Description |
|---------|-------------|
| 획일화된 디자인 | 기존 모바일 청첩장은 템플릿에 사진만 교체하는 형태 |
| 낮은 커스터마이징 | 폰트, 컬러, 레이아웃 세부 조정 불가 |
| 감성 부재 | 기능 중심 UI로 "초대"의 설렘을 전달하기 어려움 |
| 복잡한 제작 과정 | 웨딩 준비에 지친 커플에게 또 다른 부담 |

### 2.2. Target Audience
- **Primary**: 20대 후반 ~ 30대 중반 예비 부부
- **Characteristics**:
  - 심미적 기준이 높고 "나만의 취향"을 중시
  - Instagram/Pinterest에서 웨딩 레퍼런스 수집
  - "센스 있다", "너답다"는 반응을 원함
  - 복잡한 도구 없이 전문가 수준의 결과물 기대

## 3. Core Value Proposition (CVP)

### "The Canvas for Your Love Story"
단순한 정보 전달 도구가 아닌, 두 사람의 이야기를 담는 **디지털 러브레터**.

### 3.1. 7가지 핵심 차별점

| # | Feature | Value |
|---|---------|-------|
| 1 | **초개인화 (Hyper-Personalization)** | Design System 기반 커스터마이징. 타이포그래피, 컬러 팔레트, 질감까지 세밀 조정 |
| 2 | **시네마틱 경험 (Cinematic Experience)** | 몰입감 있는 인터랙션 (Fade, Parallax). 배경음악과 비주얼의 싱크 |
| 3 | **AI 초대글 작성 (AI Invitation Writer)** | 키워드 입력 시 감성적인 초대 문구 자동 생성 |
| 4 | **라이브 웨딩 모드 (Live Wedding Mode)** | 예식 당일 디지털 사이니지 모드. 방명록 실시간 롤링 |
| 5 | **시크릿 영상 편지 (Hidden Easter Egg)** | 특정 사진 길게 누르면 친구들만 볼 수 있는 시크릿 콘텐츠 |
| 6 | **하객 인사이트 (Guest Insight)** | 어떤 사진을 오래 봤는지, RSVP 응답률 패턴 분석 |
| 7 | **개인화 URL & QR (Personalized URL)** | `yeonjeong.kr/민영과지훈` 형태의 기억하기 쉬운 고유 주소 |

## 4. Go-to-Market Strategy

### 4.1. Positioning Statement
> "연정은 글솜씨가 없어도 감동적인, 디자인 전문가가 아니어도 아름다운, 세상에 단 하나뿐인 모바일 청첩장을 3분 만에 만들 수 있는 프리미엄 서비스입니다."

### 4.2. Key Messages
| Audience | Message |
|----------|---------|
| 예비 신부 | "하객들에게 '이런 청첩장 처음 봤다'는 말 들어보세요" |
| 예비 신랑 | "웨딩 준비 스트레스? 청첩장은 3분이면 끝나요" |
| 양가 부모님 | "카카오톡으로 쉽게 공유하고, 계좌번호도 한 번에" |

### 4.3. Conversion Funnel
```
Landing Page Visit
      ↓
[무료로 시작하기] CTA 클릭
      ↓
사진 1장 업로드 → 5개 테마에 자동 합성 (No-Login Try)
      ↓
회원가입 & 상세 커스터마이징
      ↓
청첩장 배포 & 공유
      ↓
"Made by 연정" 링크 → 바이럴 유입
```

## 5. Success Metrics (KPIs)

| Metric | Definition | Target |
|--------|------------|--------|
| **Activation Rate** | 방문자 대비 [시안 확인하기] 완료율 | > 30% |
| **Conversion Rate** | 시안 확인 후 회원가입 전환율 | > 15% |
| **Completion Rate** | 가입 후 청첩장 발행 완료율 | > 70% |
| **Viral Coefficient** | "Made by 연정" 클릭 통한 신규 유입률 | > 0.3 |
| **NPS** | 순추천지수 | > 50 |
