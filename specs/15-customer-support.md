# Customer Support Specification

## Overview

"연정" 서비스의 고객 지원 체계, FAQ, 문의 채널 및 환불 정책 스펙입니다.

---

## 1. Support Channels

### 1.1. 채널 구성

| Channel | Purpose | Response Time | Hours |
|---------|---------|---------------|-------|
| **FAQ/Help Center** | 셀프 서비스 | 즉시 | 24/7 |
| **카카오톡 채널** | 실시간 문의 | 1시간 이내 | 10:00~18:00 |
| **이메일** | 상세 문의, 환불 | 24시간 이내 | 24/7 접수 |
| **인앱 문의** | 빠른 문의 | 24시간 이내 | 24/7 접수 |

## 2. FAQ / Help Center

### 2.1. 카테고리 구조

| Category | Topics |
|----------|--------|
| **시작하기** | 회원가입, 로그인, 계정 관리 |
| **청첩장 만들기** | 에디터 사용법, 사진 업로드, 커스터마이징 |
| **결제/쿠폰** | 결제 방법, 쿠폰 사용, 환불 |
| **발행/공유** | 발행 방법, URL 공유, 카카오톡 전송 |
| **기능 안내** | RSVP, 방명록, 갤러리, 지도 |
| **문제 해결** | 오류 해결, 접속 문제, 수정 방법 |

### 2.2. 주요 FAQ 콘텐츠

#### 시작하기

```markdown
Q. 회원가입은 어떻게 하나요?
A. 연정은 카카오톡 또는 네이버 계정으로 간편하게 가입할 수 있습니다.
   1. [무료로 시작하기] 버튼을 클릭합니다.
   2. 카카오톡 또는 네이버 중 원하는 로그인 방식을 선택합니다.
   3. 본인 확인 후 자동으로 회원가입이 완료됩니다.

Q. 비밀번호를 잊어버렸어요.
A. 연정은 소셜 로그인(카카오/네이버)만 지원합니다.
   카카오톡이나 네이버 계정의 비밀번호를 찾아주세요.
   - 카카오: https://accounts.kakao.com/
   - 네이버: https://nid.naver.com/

Q. 탈퇴하고 싶어요.
A. [마이페이지] > [설정] > [회원 탈퇴]에서 탈퇴할 수 있습니다.
   ⚠️ 탈퇴 시 작성 중인 청첩장과 발행된 청첩장이 모두 삭제됩니다.
```

#### 청첩장 만들기

```markdown
Q. 청첩장은 몇 개까지 만들 수 있나요?
A. 청첩장은 계정당 1개만 만들 수 있습니다.
   새로운 청첩장을 만들려면 기존 청첩장을 삭제해야 합니다.

Q. 사진은 몇 장까지 업로드할 수 있나요?
A. 갤러리에 최대 30장까지 업로드할 수 있습니다.
   각 사진은 10MB 이하, JPG/PNG/WebP 형식만 지원합니다.

Q. 사진 크기나 비율에 제한이 있나요?
A. 최소 800x800 픽셀 이상의 사진을 권장합니다.
   세로 사진, 가로 사진 모두 사용 가능하며, 에디터에서 크롭할 수 있습니다.

Q. 업로드한 사진을 삭제/교체하고 싶어요.
A. [청첩장 수정] > [갤러리]에서 사진을 선택 후 삭제하거나 새 사진으로 교체할 수 있습니다.
   발행 후에도 수정이 가능합니다.
```

#### 결제/쿠폰

```markdown
Q. 가격이 얼마인가요?
A. 청첩장 발행 쿠폰 1장 기준 ₩19,900입니다.
   - 프리미엄 기능(시크릿 콘텐츠, AI 초대글 등) 모두 포함
   - 유효기간: 구매일로부터 90일
   - 1개 청첩장 발행에 1쿠폰 필요

Q. 결제 수단은 무엇이 있나요?
A. 신용카드, 체크카드, 카카오페이, 네이버페이를 지원합니다.

Q. 쿠폰 유효기간이 지나면 어떻게 되나요?
A. 유효기간이 지난 쿠폰은 자동 소멸되며 복구되지 않습니다.
   예식일 기준 충분한 여유를 두고 구매해주세요.

Q. 영수증/세금계산서를 받을 수 있나요?
A. [마이페이지] > [결제 내역]에서 영수증을 다운로드할 수 있습니다.
   세금계산서가 필요한 경우 support@yeonjeong.kr로 문의해주세요.
```

#### 발행/공유

```markdown
Q. 청첩장 URL을 바꿀 수 있나요?
A. 네, [마이페이지] > [청첩장 관리] > [URL 설정]에서 변경할 수 있습니다.
   예: yeonjeong.kr/민영과지훈
   ⚠️ 기존 URL로 공유한 링크는 새 URL로 자동 리다이렉트됩니다.

Q. 발행 후에도 내용을 수정할 수 있나요?
A. 네, 발행 후에도 언제든 수정할 수 있습니다.
   수정사항은 즉시 반영되며, 기존에 공유한 링크에서도 업데이트됩니다.

Q. 카카오톡으로 어떻게 공유하나요?
A. 청첩장 하단의 [공유하기] 버튼을 누르면 카카오톡으로 공유할 수 있습니다.
   예쁜 미리보기 이미지와 함께 전송됩니다.

Q. 발행을 취소하고 싶어요.
A. 발행 후 24시간 이내에 [마이페이지]에서 발행 취소가 가능합니다.
   취소 시 사용한 쿠폰이 복원됩니다 (1회 한정).
```

#### 기능 안내

```markdown
Q. RSVP는 어떻게 확인하나요?
A. [마이페이지] > [참석 현황]에서 참석 응답을 확인할 수 있습니다.
   - 참석/불참 인원 현황
   - 참석자 명단 및 연락처
   - 식사 여부 (설정 시)

Q. 방명록에 부적절한 글이 올라왔어요.
A. [마이페이지] > [방명록 관리]에서 해당 글을 삭제할 수 있습니다.
   방명록 기능을 비활성화할 수도 있습니다.

Q. 계좌번호 복사가 안 돼요.
A. 일부 브라우저(특히 카카오톡 인앱)에서 복사 기능이 제한될 수 있습니다.
   [Chrome/Safari로 열기] 버튼을 눌러 외부 브라우저에서 열어보세요.

Q. 지도가 안 보여요.
A. 위치 정보가 정확하게 입력되었는지 확인해주세요.
   주소 검색 후 지도에서 정확한 위치를 선택해주세요.
```

#### 문제 해결

```markdown
Q. 청첩장 페이지가 안 열려요.
A. 다음을 확인해주세요:
   1. 인터넷 연결 상태 확인
   2. 브라우저 캐시 삭제 후 재시도
   3. 다른 브라우저로 접속
   4. 문제 지속 시 support@yeonjeong.kr로 문의

Q. 이미지가 깨져서 보여요.
A. 이미지 파일이 손상되었을 수 있습니다.
   원본 이미지를 다시 업로드해주세요.
   문제 지속 시 다른 형식(JPG↔PNG)으로 변환 후 업로드해보세요.

Q. 모바일에서 레이아웃이 이상해요.
A. 기종과 증상을 캡처하여 support@yeonjeong.kr로 보내주세요.
   확인 후 빠르게 수정하겠습니다.
```

## 3. 환불 정책

### 3.1. 환불 기준

| 상황 | 환불 가능 여부 | 환불 금액 |
|------|---------------|----------|
| 쿠폰 미사용, 구매 후 7일 이내 | ✅ 가능 | 전액 |
| 쿠폰 미사용, 구매 후 7일 초과 | ❌ 불가 | - |
| 쿠폰 사용 (청첩장 발행) | ❌ 불가 | - |
| 청첩장 발행 후 24시간 이내 발행 취소 | ⚠️ 쿠폰 복원 (1회) | 쿠폰 |
| 서비스 장애로 인한 피해 | ✅ 가능 | 전액 또는 쿠폰 보상 |
| 중복 결제 | ✅ 가능 | 중복분 전액 |

### 3.2. 환불 절차

```
1. 환불 요청
   └─ 이메일: support@yeonjeong.kr
   └─ 카카오톡 채널 문의
         ↓
2. 요청 접수 확인 (24시간 이내)
   └─ 환불 가능 여부 확인
   └─ 필요 시 추가 정보 요청
         ↓
3. 환불 승인
   └─ 승인 알림 발송
         ↓
4. 환불 처리 (영업일 기준 3~5일)
   └─ 결제 수단에 따라 상이
   └─ 카드: 카드사 영업일 기준
   └─ 간편결제: 해당 서비스 기준
         ↓
5. 환불 완료 알림
```

### 3.3. 환불 요청 템플릿

```markdown
## 환불 요청 시 필요 정보

- 이메일(가입 계정):
- 주문번호:
- 결제일:
- 환불 사유:
- 환불받을 계좌 (계좌이체 결제 시):
```

### 3.4. 특수 케이스

| Case | Policy |
|------|--------|
| **예식 취소/연기** | 증빙 서류 제출 시 쿠폰 유효기간 연장 또는 환불 검토 |
| **중복 계정 구매** | 본인 확인 후 1개 계정으로 이전 또는 환불 |
| **서비스 장애 보상** | 장애 시간/영향에 따라 쿠폰 또는 환불 제공 |

## 4. 문의 응대 가이드라인

### 4.1. 응대 원칙

| Principle | Description |
|-----------|-------------|
| **공감** | 고객의 상황과 감정에 먼저 공감 표현 |
| **명확** | 쉽고 명확한 언어로 안내 |
| **신속** | 답변 가능한 건은 즉시, 확인 필요 건은 예상 시간 안내 |
| **책임** | 문제 해결까지 담당자가 끝까지 follow-up |

### 4.2. 응대 톤앤매너

```markdown
## 좋은 예시
"안녕하세요, 연정입니다 💜
문의 주신 내용 확인했어요.
청첩장 발행 후에도 언제든 수정 가능하니 걱정 마세요!
[마이페이지] > [청첩장 수정]에서 바로 수정하실 수 있습니다.
더 궁금한 점 있으시면 편하게 말씀해주세요 😊"

## 피해야 할 예시
"해당 건은 환불 불가합니다."
"약관에 명시되어 있습니다."
"확인이 필요합니다. 기다려주세요."
```

### 4.3. 상황별 응대 템플릿

#### 일반 문의

```markdown
안녕하세요, 연정입니다 💜
문의 주셔서 감사합니다.

[답변 내용]

더 궁금한 점 있으시면 편하게 말씀해주세요!
아름다운 청첩장 완성을 응원합니다 🌸
```

#### 환불 요청 (승인)

```markdown
안녕하세요, 연정입니다 💜

요청하신 환불 건 확인했습니다.
아래와 같이 환불 처리 도와드릴게요.

📌 환불 정보
- 주문번호: ORD-XXXX
- 결제금액: ₩19,900
- 예상 환불일: 영업일 기준 3~5일 이내

카드사 사정에 따라 환불 시점이 달라질 수 있는 점 양해 부탁드려요.
환불 완료 시 다시 안내드리겠습니다.

언제든 연정이 필요하시면 다시 찾아주세요 😊
```

#### 환불 요청 (불가)

```markdown
안녕하세요, 연정입니다 💜

문의 주신 환불 건 확인했어요.
말씀하신 경우는 이용약관에 따라 환불이 어려운 점 안내드려요.

📌 환불 불가 사유
- 쿠폰 사용 완료 (청첩장 발행)
- 구매 후 7일 초과

너무 아쉽지만, 발행 후 24시간 이내라면 발행 취소 후 쿠폰 복원이 가능해요!
(1회 한정) 해당하시면 말씀해주세요.

다른 문의 사항 있으시면 편하게 말씀해주세요.
```

#### 오류/버그 신고

```markdown
안녕하세요, 연정입니다 💜

앗, 불편을 드려 정말 죄송해요 😢
말씀하신 문제 확인하고 빠르게 수정할게요.

📌 확인 사항
- 증상: [고객이 설명한 문제]
- 기기/브라우저: [확인 필요 시 요청]
- 스크린샷: [있다면 감사]

혹시 아래 정보도 알려주시면 더 빨리 해결할 수 있어요:
- 사용하신 기기 (예: 아이폰 15, 갤럭시 S24)
- 브라우저 (예: 카카오톡 인앱, 크롬, 사파리)

확인되는 대로 다시 연락드릴게요!
```

#### 서비스 장애 안내

```markdown
안녕하세요, 연정입니다.

현재 일시적인 서비스 오류가 발생했습니다.
빠른 복구를 위해 최선을 다하고 있어요.

📌 현재 상황
- 발생 시각: 2025-01-07 14:30
- 영향 범위: [청첩장 조회/결제/등]
- 예상 복구: 확인 중

복구 완료 시 다시 안내드리겠습니다.
불편을 드려 정말 죄송합니다 🙏
```

## 5. 에스컬레이션 정책

### 5.1. 에스컬레이션 기준

| Level | Criteria | Handler |
|-------|----------|---------|
| **L1** | 일반 문의, FAQ로 해결 가능 | CS 담당자 |
| **L2** | 기술적 문제, 환불 검토 | 팀 리드 |
| **L3** | 심각한 불만, 법적 이슈 | 운영 책임자 |

### 5.2. 에스컬레이션 트리거

| Trigger | Action |
|---------|--------|
| 고객 2회 이상 동일 문의 | L2 에스컬레이션 |
| 환불/보상 요청 | L2 에스컬레이션 |
| 욕설/협박 | L3 에스컬레이션 + 증거 보존 |
| 법적 조치 언급 | L3 에스컬레이션 |
| SNS/커뮤니티 부정 게시물 | L2 모니터링 + 대응 |

## 6. 고객 피드백 수집

### 6.1. 피드백 채널

| Channel | Type | Frequency |
|---------|------|-----------|
| **인앱 피드백** | 별점 + 코멘트 | 발행 완료 후 |
| **NPS 설문** | 추천 점수 | 예식일 +1주 |
| **문의 후 만족도** | 별점 | 문의 종료 후 |
