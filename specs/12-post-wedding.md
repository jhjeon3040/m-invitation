# Post-Wedding Features Specification

## Overview

예식 이후에 사용하는 기능들을 정의합니다.
청첩장이 "감사장"으로 자연스럽게 전환되는 경험을 제공합니다.

---

## Feature Summary

| Feature | Trigger | Description |
|---------|---------|-------------|
| **감사 인사** | 예식일 +1일 | 하객들에게 감사 메시지 발송 |
| **청첩장 → 감사장 전환** | 수동 또는 자동 | 청첩장 콘텐츠를 감사장으로 변경 |
| **웨딩 포토 공유** | 수동 | 본식 사진 갤러리 추가 |
| **청첩장 아카이브** | 예식일 +1년 | 비공개 전환 및 보관 |

---

## 1. Thank You Message (감사 인사)

### Trigger
- 예식일 다음 날 대시보드에 안내 배너 표시
- 또는 사용자가 직접 "감사 인사 보내기" 클릭

### Flow
```
[대시보드]
    ↓
"예식은 잘 마치셨나요? 감사 인사를 보내보세요" 배너
    ↓
[감사 인사 보내기] 클릭
    ↓
┌─────────────────────────────────────────────────────────────┐
│  감사 인사 작성                                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  감사 메시지                                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 따뜻한 축하와 격려를 보내주신 모든 분들께            │   │
│  │ 진심으로 감사드립니다.                               │   │
│  │                                                     │   │
│  │ 보내주신 사랑 잊지 않고,                             │   │
│  │ 더 예쁘게 잘 살겠습니다. 💕                         │   │
│  │                                                     │   │
│  │                         - 민영 & 지훈 드림          │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  [AI로 작성하기]                                             │
│                                                             │
│  ─────────────────────────────────────────────────────────  │
│                                                             │
│  사진 추가 (선택)                                            │
│  ┌──────┐ ┌──────┐ ┌──────┐                                │
│  │ +추가 │ │      │ │      │    최대 3장                    │
│  └──────┘ └──────┘ └──────┘                                │
│                                                             │
│  ─────────────────────────────────────────────────────────  │
│                                                             │
│  발송 대상                                                   │
│  ○ RSVP 응답자 전체 (45명)                                  │
│  ○ 참석자만 (38명)                                          │
│  ○ 방명록 작성자 포함 (52명)                                │
│                                                             │
│              [미리보기]        [발송하기]                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Delivery Methods

| Method | Description | Requirement |
|--------|-------------|-------------|
| **카카오톡** | 알림톡 발송 | 연락처 필수 |
| **SMS** | 문자 발송 | 연락처 필수 |
| **청첩장 내 표시** | 청첩장 상단에 감사 배너 | 연락처 불필요 |

### Default: 청첩장 내 표시
- 연락처가 없는 경우에도 감사 인사 가능
- 청첩장 재방문 시 감사 메시지 확인

## 2. Invitation → Thank You Card (청첩장 → 감사장 전환)

### Transformation
청첩장의 일부 섹션을 감사장에 맞게 변경합니다.

| 섹션 | 청첩장 | 감사장 |
|------|--------|--------|
| Hero | "결혼합니다" | "감사합니다" |
| 초대글 | 초대 문구 | 감사 문구 |
| RSVP | 참석 여부 폼 | 숨김 |
| 계좌 | 계좌 정보 | 숨김 또는 유지 (선택) |
| 갤러리 | 웨딩 화보 | 웨딩 화보 + 본식 사진 |
| 방명록 | 축하 메시지 | 축하 메시지 (읽기 전용) |

### Trigger Options

| Option | Description |
|--------|-------------|
| **수동 전환** | 대시보드에서 "감사장으로 전환" 버튼 |
| **자동 전환** | 예식일 +7일 후 자동 제안 |
| **전환 안 함** | 청첩장 그대로 유지 |

### UI

```
┌─────────────────────────────────────────────────────────────┐
│  감사장으로 전환하기                                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  청첩장을 감사장으로 전환하면:                                │
│                                                             │
│  ✓ "결혼합니다" → "감사합니다" 변경                          │
│  ✓ 초대글 → 감사 문구로 변경                                 │
│  ✓ RSVP 폼 숨김                                             │
│  ✓ 본식 사진 추가 가능                                       │
│                                                             │
│  ⚠️ 전환 후에도 언제든 원래대로 돌릴 수 있어요               │
│                                                             │
│          [나중에]          [감사장으로 전환]                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 3. Wedding Photo Sharing (본식 사진 공유)

### Purpose
스튜디오에서 받은 본식 사진을 하객들과 공유합니다.

### Features

| Feature | Description |
|---------|-------------|
| 별도 갤러리 | "본식 스냅" 탭 추가 |
| 업로드 | 기존 갤러리와 동일 (최대 50장) |
| 다운로드 허용 | 하객이 사진 저장 가능 (선택) |
| 공개 범위 | 전체 / RSVP 응답자만 / 비공개 |

### Gallery Structure (전환 후)

```
┌─────────────────────────────────────────────────────────────┐
│  📸 갤러리                                                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [웨딩 화보]    [본식 스냅]                    ← 탭 전환      │
│      ↓              ↓                                       │
│  기존 사진들     새로 추가된 본식 사진                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Photo Download (Optional)

```
┌─────────────────────────────────────────────────────────────┐
│  사진 다운로드 설정                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [ ] 하객이 사진을 저장할 수 있게 허용                        │
│                                                             │
│  허용 시:                                                    │
│  - 각 사진에 다운로드 버튼 표시                               │
│  - 워터마크 추가 (선택)                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 4. Archive & Expiration (아카이브 및 만료)

### Timeline

```
예식일
  │
  ├─ +1일: "감사 인사 보내기" 안내
  │
  ├─ +7일: "감사장으로 전환" 제안
  │
  ├─ +30일: "본식 사진 추가하기" 리마인더
  │
  ├─ +11개월: "청첩장 만료 예정" 알림
  │
  └─ +12개월: 자동 비공개 전환
```

### Expiration Policy

| 시점 | Action | 복구 |
|------|--------|------|
| 예식일 +11개월 | 만료 예정 알림 (이메일) | - |
| 예식일 +12개월 | 자동 비공개 전환 | 대시보드에서 공개 가능 |
| 예식일 +24개월 | 아카이브 (접근 제한) | 고객센터 문의 |
| 예식일 +36개월 | 데이터 삭제 | 불가 |

### Archive View

비공개된 청첩장도 본인은 열람 가능:

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  🔒 이 청첩장은 비공개 상태입니다                             │
│                                                             │
│  만료일: 2026.05.16                                         │
│                                                             │
│  [다시 공개하기]    [PDF로 저장하기]                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### PDF Export

만료 전 청첩장을 PDF로 보관할 수 있는 기능:

| Feature | Description |
|---------|-------------|
| 포맷 | A4 세로, 고해상도 |
| 내용 | 모든 섹션 포함 |
| 사진 | 원본 화질 |
| 방명록 | 모든 메시지 포함 |

## UX Principle: Non-Intrusive

### 핵심 원칙
> "예식 후 기능은 **제안**하되, **강요**하지 않는다"

### Implementation

| Do | Don't |
|----|-------|
| 배너로 안내 | 팝업으로 강제 표시 |
| "나중에" 버튼 제공 | 무조건 전환 |
| 1회만 안내 | 반복 알림 |
| 원복 가능 명시 | 되돌릴 수 없다고 표시 |

### Progressive Disclosure 적용

```
[기본 상태 - 예식 후]

대시보드 상단에 작은 배너:
┌─────────────────────────────────────────────────────────────┐
│ 💌 예식 잘 마치셨나요? 감사 인사를 보내보세요   [보내기] [닫기] │
└─────────────────────────────────────────────────────────────┘

- "닫기" 누르면 30일간 표시 안 함
- 강제가 아닌 선택
```
